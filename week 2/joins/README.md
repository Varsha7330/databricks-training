# SQL JOINS Assignment

## Description
This project contains SQL JOIN practice queries implemented using MySQL. The assignment focuses on understanding and applying different JOIN operations to retrieve related data from multiple tables.

The project includes:
- Table creation queries
- Insert statements
- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- FULL OUTER JOIN
- SELF JOIN
- Aggregate functions with JOINs
- Employee, Department, Project, Salary, and Contact analysis
