# SQL REGEX Foundation Assignment

## Description
This project contains basic to intermediate SQL REGEX concepts implemented using MySQL. The assignment focuses on understanding Regular Expressions (REGEX) from fundamentals and applying pattern matching techniques for data extraction and validation.

The project includes:
- REGEX basics
- Digit extraction
- Alphabet extraction
- Pattern matching
- Email validation
- Phone number validation
- String extraction techniques
- Data quality and validation concepts

---

## Topics Covered

### REGEX Concepts
- Character matching
- Digit extraction
- Alphabet extraction
- Start of string (^)
- End of string ($)
- Exact character matching using {}
- Continuous matching using +
- Reserved characters
- Escaping special characters

### REGEX Patterns
- [0-9]
- [A-Za-z]
- [0-9]+
- [A-Za-z]+
- [0-9]{2}
- [A-Za-z]{2}
- ^ pattern
- $ pattern
