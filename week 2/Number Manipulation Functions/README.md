
# SQL Number Manipulation Functions Assignment

## Description
This project contains SQL practice queries based on numeric and mathematical functions in MySQL. The assignment focuses on performing number manipulation, calculations, rounding operations, formatting, and mathematical computations using built-in SQL functions.

The project includes:
- Table creation queries
- Insert statements
- Numeric functions
- Salary calculations
- Tax calculations
- Real-time mathematical operations
- Data formatting queries
