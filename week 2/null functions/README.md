# SQL NULL Functions Assignment

## Description
This project contains SQL practice queries based on NULL handling functions in MySQL. The assignment focuses on understanding how NULL values are managed in databases using different SQL functions and conditions.

The project includes:
- Table creation queries
- Insert statements
- NULL handling functions
- Conditional queries
- Real-time business scenarios
- Advanced SQL NULL operations

---

## Topics Covered

### NULL Functions
- IS NULL
- IS NOT NULL
- IFNULL()
- COALESCE()
- NULLIF()

### Concepts Practiced
- Replacing NULL values
- Handling missing data
- Avoiding divide-by-zero errors
- Default value handling
- Conditional NULL replacements
- Salary and bonus calculations
- Product and order analysis

## Learning Outcomes
After completing this assignment, I learned:
- How NULL values work in SQL
- How to replace NULL values using IFNULL()
- How COALESCE() returns the first non-null value
- How NULLIF() is used in conditional comparisons
- How to filter NULL values using IS NULL and IS NOT NULL
- How to handle real-world missing data scenarios using SQL queries
