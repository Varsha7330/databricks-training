# SQL REGEX Assignment

## Description
This project contains SQL practice queries based on Regular Expressions (REGEX) in MySQL. The assignment focuses on extracting, filtering, and manipulating string patterns using REGEX functions and pattern matching techniques.

The project includes:
- Table creation queries
- Insert statements
- REGEX pattern matching
- String extraction queries
- Email pattern analysis
- Phone number extraction
- Numeric and alphabetic pattern handling
