# SQL Practice Assignment

**Date:** 14/07/2026

## Assignment Overview

This assignment focuses on practicing fundamental SQL concepts using the `Employees` table. The objective is to improve query-writing skills by working with different SQL clauses and operators.

## Topics Covered

* SELECT
* WHERE
* GROUP BY
* HAVING
* DISTINCT
* TOP / LIMIT
* Comparison Operators (`=`, `>`, `<`, `>=`, `<=`, `<>`)
* Logical Operators (`AND`, `OR`, `NOT`)
* `IN` and `NOT IN`
* `BETWEEN` and `NOT BETWEEN`
* `LIKE` Operator

## Table Used

**Employees**

Columns:

* `emp_id`
* `emp_name`
* `department`
* `salary`
* `city`
* `experience`

## Tasks Completed

* Retrieved employee details using `SELECT`.
* Filtered records using `WHERE`.
* Performed aggregation using `GROUP BY`.
* Filtered grouped results using `HAVING`.
* Retrieved unique values using `DISTINCT`.
* Displayed top records using `LIMIT` (MySQL).
* Applied comparison operators for conditional filtering.
* Used logical operators (`AND`, `OR`, `NOT`).
* Filtered records using `IN` and `NOT IN`.
* Queried ranges using `BETWEEN`.
* Performed pattern matching using the `LIKE` operator.

## Learning Outcomes

By completing this assignment, I learned how to:

* Retrieve and filter data from a database.
* Group records and perform aggregate calculations.
* Filter grouped data using `HAVING`.
* Use SQL operators to create complex queries.
* Search data using pattern matching with `LIKE`.
* Write MySQL-compatible queries using `LIMIT`.
