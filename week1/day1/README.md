# Week 1 - Day 1
## 📚 Topics Covered
* Basic SQL queries using `SELECT`
* Filtering data using `WHERE` conditions
* Working with comparison operators (>, <, >=, <=)
* Retrieving specific columns from a table
* SQL Joins (INNER JOIN between Employee and Department)
* Pattern matching using `LIKE`
  * Starts with (`J%`)
  * Ends with (`%e`)
  * Contains (`%a%`)
  * Single character wildcard (`_`)
* String functions (`LENGTH`)
* Date functions
  * `YEAR()`
  * `MONTH()`
  * `CURDATE()`
* Date comparisons and intervals
* Aggregate functions
  * `SUM()`
  * `AVG()`
  * `MIN()`
  * `MAX()`
  * `COUNT()`
* Grouping data using `GROUP BY`
* Filtering grouped data using `HAVING`
* Sorting results using `ORDER BY`
* Combining multiple clauses (GROUP BY + HAVING + ORDER BY)
---
## 📝 Summary
Learned and practiced fundamental SQL concepts including data retrieval, filtering, aggregation, joins, and sorting. This forms the foundation for working with Databricks and data engineering tasks.
